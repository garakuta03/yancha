## 0. 前提: ポリシーの現状（公式情報）

- YouTubeは**2025年7月15日**に「繰り返しの多いコンテンツ（repetitious content）」ポリシーを「**量産型のコンテンツ（inauthentic content）**」に改名。公式ヘルプの収益化不可の例: 「類似した内容や繰り返しの多いコンテンツで教育的価値が不十分なもの」「説明・解説・教育的価値が最小限または皆無の画像スライドショー」「汎用的テンプレートから大量生産されたAI生成コンテンツ」等。
  - 公式: https://support.google.com/youtube/answer/1311392 （日本語版で直接確認）
- YouTube側（クリエイターリエゾン Rene Ritchie）は「**マイナーな更新**であり、量産・繰り返しコンテンツは以前から収益化不可。reused contentポリシーに変更はない」と明言（メディア報道）。
  - TechCrunch 2025-07-09: https://techcrunch.com/2025/07/09/youtube-prepares-crackdown-on-mass-produced-and-repetitive-videos-as-concern-over-ai-slop-grows/
  - Search Engine Journal: https://www.searchenginejournal.com/youtube-clarifies-monetization-update-targeting-spam-not-reaction-channels/550755/

## 1. 審査落ち・剥奪の具体事例（2025-2026中心）

**確認できた一次事例は少数で、いずれも「一事例であり一般化不可」:**

1. **【一次体験談・日本】AI音楽チャンネル「MAYU」**（登録7,200人超）: 2025年秋、順調に収益が出ていた中で突然収益化停止。YouTubeの提示理由は「**再利用されたコンテンツ**」と「**繰り返しの多いコンテンツ**（=量産型に改名済み）」の2つ。「すべてAIで自作したオリジナル」と制作過程を示す再審査動画を提出したが**却下**。映像はAI静止画+雪・レンズフレア等のエフェクトのみだった（記事2025-10-24公開）。
   - https://note.com/grand_yarrow659/n/n3ee0c3e7908c
2. **【一次情報・日本】登録27.5万人の睡眠ヒーリング音楽チャンネル**（チャンネル売買サイト掲載）: 2022年5月に「再利用されたコンテンツ」判定で収益化剥奪。**複数回の復帰申請・再申請がすべて却下**され、収益ゼロのまま300万円で売りに出された。※2022年の事例で2025-26改定より前だが、このジャンルでの剥奪→復帰困難の実例。
   - https://rakkoma.com/project/detail/4945
3. **【伝聞・英語圏】** The Next Web報道内で、クリエイターDoctor NOS（170万登録）が「顔出しなしで自分と同種のコンテンツをやっている人の**ほとんどが収益化を剥奪されている**」と発言（2026年、本人の観測であり統計ではない）。
   - https://thenextweb.com/news/youtube-ai-slop-crackdown-faceless-creators-collateral-damage
4. **【業界記事・チャンネル名なし】** 業界系記事は「ambient/lofi/瞑想音楽はYPP却下率が最も高い部類。長尺・映像変化の乏しさが繰り返し判定を誘発」「30秒のループ素材を1時間に伸ばした雨音・ホワイトノイズ動画の大量投稿が典型的な却下パターン」と一致して指摘するが、**個別のチャンネル名を挙げた検証可能な事例は提示していない**。
   - Dynamoi: https://dynamoi.com/learn/youtube-music-promotion/ambient-music-channel-monetization-rules
   - vidIQ（2026-03公開/06更新。「雨音・ホワイトノイズのループ」を却下トリガーとして明示）: https://vidiq.com/blog/post/types-youtube-channels-dont-get-monetized/
   - InVideo: https://invideo.io/blog/youtube-kills-ai-faceless-channels/

**注意:** Reddit（r/PartneredYoutube等）の環境音チャンネル剥奪スレッドは複数の検索クエリ（英日）で特定できず。存在しないのではなく、検索インデックスとアクセス制限の問題の可能性が高い（限界事項）。

## 2. 審査に通っている環境音・アンビエントチャンネルの条件（業界の定説、二次情報）

- **自作の証明**: 説明欄に「オリジナル制作」と明記、制作過程の記録（DAW/プロジェクトファイル、フィールド録音の実写映像）を残す。
  - UREBAラボ: https://ureba.jp/lab/youtube-sleep-bgm-earnings/
- **動画間のバリエーション**: 「審査通過にはdemonstrable variation（動画間の実証可能な差異）が必要」（Dynamoi）。映像を毎回変える、タイトルの連番テンプレ（"Lofi Mix #47"型）を避ける。
- **人間の創作関与の可視化**: 長尺トラックの合間にbehind-the-scenes・制作解説・アーティストコメンタリー動画を混ぜる。
- **審査官は人気動画を重点チェック**するため、最も再生されている動画にオリジナル要素を強く入れる（ロロント: https://roronto.jp/mkt/youtube/failed-the-monetization-review/ ）。
- **代替経路**: アグリゲーター経由のArt Tracks（YouTube Music）はチャンネルのYPP審査なしで音源収益化が可能（Dynamoi FAQ: https://dynamoi.com/learn/faqs/ambient-music-channel-monetization-rules ）。
- 実例（一次・成功）: 収益化済みBGMチャンネル運営者のJUN BLOGは「収益化まで6ヶ月。ニッチ特化（和風Lo-Fi等）とブランド統一、テロップ・効果音等の編集投資が必要。審査通過後の停止例も多い」: https://jusan-blog.com/ai-trap/ （2024-07、改定前）

## 3. 「量産・テンプレート」判定の回避策と再申請の通し方（コミュニティ・コンサルの定説）

- **再申請手順の定説**（vidIQ, 2026年更新）: 問題動画を削除/非公開 → **明確にオリジナルな動画を5〜10本追加** → 初回却下後30日（2回目以降90日）待って再申請。再審査は2〜4週間。
- **アピール（人間審査）で示すもの**（Lenostube: https://www.lenostube.com/en/how-to-appeal-youtube-demonetization/ 、AwesomeCreatorAcademy: https://www.awesomecreatoracademy.com/blog/youtube-reused-content-guide ）:
  - 編集ソフト（Premiere/DaVinci等）の画面を録画し、**音声・映像トラック構成を見せて工数を証明**
  - アピール動画の内容とチャンネル上の実際の動画を紐付ける
  - 初期動画がテンプレ的だった場合は「改善の経緯」を説明
  - 公式のアピール窓口: https://support.google.com/youtube/answer/9564590
- **判定回避のシグナル管理**（SubSub 2026-05: https://www.subsub.io/blog/inauthentic-content-why-so-many-creators-are-getting-demonetized 、ScaleLab: https://scalelab.com/en/why-youtube-is-cracking-down-on-ai-generated-content-in-2026 ）:
  - メタデータ（タイトル・サムネ・説明）の多様化、同一構造の連発回避
  - **投稿頻度を抑える**（1日十数本はbot判定シグナル。2026年は動画単位でなくチャンネル単位で量産検出との指摘）
  - 静止画1枚+音声のみを避け、動きのある映像へ
  - 台本・制作ノート・プロジェクトファイルを保存し関与を文書化
- **注意（MAYU事例との矛盾）**: 「制作過程動画を出せば通る」は定説だが、上記1の一次体験談では制作過程を示しても却下されており、**確実な方法は存在しない**。

## 4. 2025年7月改定後、このジャンルで大量剥奪は起きたか

**結論: 「環境音・睡眠系を狙い撃ちした大量剥奪」の確証は見つからなかった。ただし執行は段階的に強化。**

- 改定自体は公式に「マイナー更新」であり、直後の大量剥奪報道はなし（TechCrunch、SEJ、Social Media Today: https://www.socialmediatoday.com/news/youtube-clarifies-monetization-update-inauthentic-repeated-content/752892/ ）。
- **2026年1月の大規模執行**（16チャンネル・計3,500万登録・47億再生を停止/削除）は、CuentosFacinantes（ドラゴンボール系AIアニメ）、Super Cat League等の**物語・アニメ系AIスロップが対象で、環境音・睡眠音楽系は含まれていない**（メディア報道: IBTimes UK https://www.ibtimes.co.uk/youtube-cracks-down-ai-generated-content-1800213 、xda-developers https://www.xda-developers.com/youtube-just-deleted-over-4-7-billion-views-worth-ofai-slop-videos/ ）。
- 日本の「収益化停止ラッシュ/BAN祭り」（2026年1月〜）の対象もAI量産動画・大食い・フィットネス・家族系が中心で、睡眠BGM・環境音は主要リスト外（リアルサウンド via Yahoo!ニュース 2026-04-19: https://news.yahoo.co.jp/articles/fa16185d2826e5138a2240231288145dae257f41 、げげろぐ 2026-03-13: https://gegelog.com/youtube20260131 ）。2025年10月に「連鎖停止ルール」（同一運営者の別チャンネルも連帯BAN）が施行された点は要注意。
- 一方、OutlierKit（業界記事）は「2025年末以降、**純AI音楽チャンネルの40%以上がYPP失格または停止**」と主張するが、**一次出典が示されておらず未検証**（https://outlierkit.com/resources/ai-generated-music-youtube-monetization-2026/ ）。
- **推測**: ジャンル全体の一斉剥奪ではなく、「静止画+ループ音声の量産型」への審査厳格化・入口（YPP審査）での却下増という形で効いている、というのが複数ソースの整合的な解釈。

## 5. AI生成の睡眠/ヒーリング系チャンネルの成功・失敗事例

- **失敗（一次）**: MAYU氏（上記1）。AI静止画+AI音楽、2025年秋に停止、再審査却下。
- **成功主張（一次だが宣伝性あり・検証不能）**:
  - おしょー氏: Lo-Fi系AI音楽で「1ヶ月で収益化。静止画+軽いエフェクトのみ」（2025-03-24=**改定前**。有料note販売への誘導あり）: https://note.com/osyoai/n/n7a5cdcce6030
  - 「AI×睡眠用BGMで月90万?!」等のnote/Brain系情報商材が多数存在するが、いずれも販売目的で収益スクショの検証不可。
- **業界記事の見立て**: AI睡眠・瞑想系はRPM $3〜8の「まだ機能するニッチ」だが、条件は「AI開示タグ+オリジナル映像+人間のキュレーション」（OutlierKit、未検証）。Suno系ガイドの一致した見解は「**音声のみ+静止画1枚は却下、権利を持つオリジナル映像や人間の編集を加えれば承認され得る**」「完全AI生成音楽（無加工)はContent ID対象外」（Jack Righteous: https://jackrighteous.com/en-us/blogs/ai-music-distribution-guide/youtube-ai-policy-for-suno-creators 、makebestmusic: https://makebestmusic.com/blog/can-you-monetize-suno-ai-music-on-youtube ）。
- 参考（非AI・人力の成功例）: Lofi Girl、Soothing Relaxation等の大手は自作曲+独自映像で長年収益化継続。「Meditation & Relaxation - Music」（44.8万登録）は推定年収$129K（networthspot推定値、精度低）。

## yanchaプロジェクトへの示唆（推測を含む）

1. 本プロジェクトの方針（完全オリジナル・ライセンス証跡・人間レビュー）は、確認された却下パターン（ループ量産・静止画+音声・テンプレ連番・出所不明素材）の裏返しであり、方向性として妥当。
2. 最大リスクは「オリジナルでも量産型に見える」こと。一次事例（MAYU）は制作証明を出しても復活できなかった。動画ごとの映像・構成のバリエーション、制作過程の公開動画、投稿頻度の抑制、メタデータの非テンプレ化を最初から設計に組み込むべき。
3. 環境音ジャンル狙い撃ちの大量剥奪は起きていないが、YPP**入口審査の却下率がジャンル的に高い**のは複数ソースが一致。ナレーション皆無より、制作解説など人間の関与が見える動画を混ぜる構成が定説上有利。
