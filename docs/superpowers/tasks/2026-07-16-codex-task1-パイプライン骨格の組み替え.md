## Step 2: 決定論PRNG と seed導出を `@yancha/core` に追加

`packages/core/src/rng.ts` を新規作成:
```typescript
// 文字列seedから決定論的な擬似乱数生成器を作る（Math.random禁止の受け皿）
export function createRng(seed: string): () => number;
// videoIdと用途から用途別seedを導出する（音と映像で別系列にする）
export function deriveSeed(videoId: string, purpose: string): string;
```

実装方針:
- `deriveSeed`: `node:crypto` の `createHash("sha256")` に `` `${videoId}:${purpose}` `` を通し、
  hexダイジェストを返す（seed文字列そのもの）。
- `createRng`: 受け取った seed 文字列を `createHash("sha256")` で32bit整数群に落とし、
  xorshift128 等の決定論PRNGの内部状態として使う。返り値は `[0, 1)` の浮動小数点数を返す関数
  （`Math.random()` と同じインターフェース）。

**テスト**（`packages/core/tests/rng.test.ts` 新規）:
- 同一seedで `createRng` を2回呼び、それぞれから値を取り出すと**同一列**になること
- 異なる `purpose` で `deriveSeed` を呼ぶと**異なる**seedになること（同一 `videoId` でも）
- `packages/core/src/index.ts` から `createRng` / `deriveSeed` を re-export すること:
  ```typescript
  export { createRng, deriveSeed } from "./rng.js";
  ```
